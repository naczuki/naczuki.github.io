# Getting started
1. Click the "Use this template" button at the top of this repository page.
2. Set repository name to `<your-github-username>.github.io`.
3. Edit the `.well-known/nostr.json` file to set your Nostr public key(hex format).
4. (Optional) Add a relays field to the `nostr.json` file to specify your preferred relays.
5. Set your NIP-05 address to `_@<your-github-username>.github.io` in your favorite client.
